# DEMONSTRASI-CONVOLUTIONAL-NEURAL-NETWORK
DEMONSTRASI CONVOLUTIONAL NEURAL NETWORK (CNN) UNTUK KLASIFIKASI CIFAR IMAGE

Convolutional Neural Network (CNN) merupakan metode deep learning yang efektif untuk
pengenalan gambar. Pada laporan ini dilakukan demonstrasi training CNN dengan dataset
CIFAR-10, yang berisi 60.000 gambar berwarna ukuran 32×32 piksel dalam 10 kelas. Citra diresize menjadi 64×64 piksel agar model CNN dapat mengekstraksi fitur lebih mendalam.
